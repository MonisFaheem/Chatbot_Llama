This project is created on python using streamlit framwork which is very useful for creating simple UI, we also used
replicate library which is reponsible for providing API key for lama2 model.

*step 01
after opening project folder in pycharm or vs code first you have to install required librarries by runing command "pip install -r requirement.txt"
*step 02
to run this project type "streamlit run app.py" command in py charm terminal
